Part 2 · Database Fundamentals
Tables, Rows, Columns

A database holds one or more tables. Each table is like a spreadsheet:

  • Columns = fields (name, email, age)
  • Rows = records (one student, one product)
  • Primary key = a unique ID for each row (usually id)
Common Data Types

text — strings  ·  int8 / int4 — whole numbers  ·  bool — true/false  ·  timestamp — date + time  ·  uuid — unique ID

Part 3 · SQL — The Language of Databases
What is SQL?

SQL (Structured Query Language) is how we talk to databases. Four main commands:

  • SELECT — read data
  • INSERT — add rows
  • UPDATE — change rows (next lesson)
  • DELETE — remove rows (next lesson)
INSERT — Adding Data
INSERT INTO students (name, grade, gpa)
VALUES ('Jordan', 11, 3.8);
