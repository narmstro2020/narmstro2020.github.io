CUSTOM SCORE FORMULA
Touches: Game.start_new_round   Game.check_collisions
DIFFICULTY ⭐

Right now the game uses two scoring moments: a flat 100 × round per catch and a time-based round completion bonus. You'll redesign both to create a scoring system that feels like your own.

Hint: The current bonus formula is int(10000 * round_number / (1 + round_time)). A fast round on round 5 is worth a lot; a slow round on round 1 is worth almost nothing. Think about what you want to reward before you start changing numbers.
  • Change the per-catch score in check_collisions. The current formula scales with round number — you could make it flat, add a streak multiplier, or scale with time remaining instead. 
    # In check_collisions — find this line:
self.score += 100 * self.round_number  # original

# Ideas to try:
self.score += 500                         # flat 500 per catch
self.score += 200 * self.round_number      # double the scaling
self.score += max(50, 300 - self.round_time * 5) # speed bonus per catchcheck_collisions
  • Change the round completion bonus in start_new_round. Try adjusting the constants, or replace the formula entirely. 
    # In start_new_round — find this line:
self.score += int(10000 * self.round_number / (1 + self.round_time))

# Ideas to try:
self.score += max(0, 5000 - self.round_time * 25)  # flat ceiling
self.score += self.round_number * 1000             # ignore time entirely
self.score += int(20000 * self.round_number / (1 + self.round_time))start_new_round
  • Per-catch score is different from the original 100 × round
  • Round completion bonus uses a formula you designed
  • Score on HUD reflects your changes as you play

WARP UPGRADE
Touches: Player.warp   Game.start_new_round
DIFFICULTY ⭐⭐

Currently, SPACE warps the player to the exact center-bottom of the screen every time. This is safe but predictable. You'll upgrade the warp so it teleports the player to a random safe X position along the bottom — making it more exciting and less of a guaranteed escape hatch.

  • In Player.warp, replace the fixed centerx with a random x position. The player must stay inside the window bounds. 
    def warp(self):
    if self.warps > 0:
        self.warps -= 1
        self.warp_sound.play()
        self.rect.bottom = WINDOW_HEIGHT

        # Original: always center
        # (no x change — player keeps current x)

        # New: random safe x position
        self.rect.centerx = random.randint(
            self.rect.width // 2,
            WINDOW_WIDTH - self.rect.width // 2
        )Player.warp
  • Adjust the starting warp count and the warp refill per round to match your new design. More warps = easier game; fewer = more tension.

HIGH SCORE TRACKER
Touches: Game.__init__   Game.reset_game   Game.draw
DIFFICULTY ⭐⭐

The game currently forgets your best score the moment you reset. You'll add a persistent high score that survives game resets — it only goes up, never down — and display it in the HUD alongside the current score.

  • Add self.high_score to Game.__init__. It starts at 0 and never gets reset to 0 — that's the key. 
    # In Game.__init__ — add with the other score values
self.score = 0
self.high_score = 0__init__
  • In Game.reset_game, update the high score before resetting the current score to 0. 
    def reset_game(self):
    # Save the high score BEFORE wiping the current score
    if self.score > self.high_score:
        self.high_score = self.score

    self.score = 0          # this resets — high_score does not
    self.round_number = 0
    self.player.lives = 5
    self.player.warps = 2
    self.player.reset()
    self.start_new_round()reset_game
