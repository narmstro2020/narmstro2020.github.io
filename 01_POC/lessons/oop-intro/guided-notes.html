What is OOP?

11:09 ‚Äì 11:19 (10 min)

üë®‚Äçüè´ Instructor Demo

OOP organizes code into classes (blueprints) and objects (real things). You've already been using objects ‚Äî every list, string, and integer in Python is an object!

# You've been using objects this whole time! my_list = [1, 2, 3] my_string = "Hello World" print(type(my_list)) # <class 'list'> print(type(my_string)) # <class 'str'>

üí° Key Concept

A class is a blueprint (like a character creation screen). An object is a real thing made from that blueprint (like each character you create). One class ‚Üí many objects, each with their own data.

The __init__ Constructor

11:29 ‚Äì 11:39 (10 min)

üë®‚Äçüè´ Instructor Demo

__init__ (pronounced "dunder init") is a special method that runs automatically when you create a new object. It sets up the starting values.

class Hero: """A brave RPG hero character.""" def __init__(self, name, health, attack_power): self.name = name self.health = health self.attack_power = attack_power self.is_alive = True # default value # Python calls __init__ for you: aria = Hero("Aria", 100, 25) print(f"{aria.name}: {aria.health} HP")

The self Keyword

11:44 ‚Äì 11:54 (10 min)

üë®‚Äçüè´ Instructor Demo

self means "this specific object." Python fills it in automatically ‚Äî you never pass it yourself. Each object's self points to itself.

# Each object is independent: hero_a = Hero("Aria", 100, 25) hero_b = Hero("Kael", 120, 30) hero_a.health = 50 # Aria took damage print(hero_a.health) # 50 print(hero_b.health) # still 120!

Instance Attributes

11:54 ‚Äì 12:04 (10 min)

üë®‚Äçüè´ Instructor Demo

Attributes come from two places: parameters (user provides the value) and defaults (set inside __init__, same starting value for everyone).

class Monster: def __init__(self, name, monster_type, max_health): # From parameters: self.name = name self.monster_type = monster_type self.max_health = max_health # Default values: self.current_health = max_health self.level = 1 self.is_defeated = False self.loot = [] goblin = Monster("Grimjaw", "Goblin", 50)
